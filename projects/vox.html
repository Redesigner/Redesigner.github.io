Vox

What I worked on:

  • GLTF importer
  • Skeletal mesh animation system
  • OpenGL backend
  • Compute shader mesh generation
  • PBR material lighting model

A game with custom engine and editor tools -- build voxel worlds, platform, and fight enemies! The game engine uses SDL3 for handling windows, input events, and filesystem i/o. Vox currently supports the openGl API. It is currently a work in progress, but you can check the progress on the GitHub page below.
