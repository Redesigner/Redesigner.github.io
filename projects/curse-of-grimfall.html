Curse of Grimfall

What I worked on:

  • Solo Development
  • Programming
  • Character Art
  • Animation
  • Level Design

A 3D Zelda-like game written in Unreal Engine 5. I worked on all of the character design, animations, and programming. Sounds were composited from different recordings and Creative Commons sound files. Includes a demo level. Features include:

  • Target lock-on system
  • Special hitboxes for attacking and blocking - integrated with Unreal's animation editor
  • Slot based equipment
  • Full block pushing physics
  • Data based buttons and switches
  • Custom cameras for animated cutscenes
Source code is available on github below, as well as the packaged *.zip. (Currently Windows only!)
