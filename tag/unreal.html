Tether

What I worked on:

  • C++ programming
  • Physics design and programming
  • Blueprint scripting
  • Game Design
  • Character Art
  • 3D Modeling
  • Animation
  • Sound Effects

Tether was originally a game jam entry for the GMTK 2021 Game Jam. It is powered by Unreal Engine 4. Using the built-in raycasting and shape tracing functions of the engine, I programmed the collision and movement. I took special care to make sure that the movement mechanics and physics performed in a very specific manner, tweaking it so that everything feels smooth and fun. I recommend that you try out the platforming physics for yourself.
